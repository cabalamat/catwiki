Python source code:

```python
def convertQuickLinks(s):
    """ Converts [[xxx]] -> [xxx](xxx)
    @param s [str] containing markdown source
    @return [str]
    """
    QUICKLINK_RE = r"\[\[([A-Za-z0-9_ ]+)\]\]"
    REPLACE_WITH = r"[\1](\1)"
    r = re.sub(QUICKLINK_RE, REPLACE_WITH, s)
    return r
```
